# ☕ Coffee RH

Sistema web simples de **registro de ponto virtual**, onde usuários podem acessar sua conta, visualizar informações de perfil e registrar o horário de trabalho.

---

## 📌 Sobre o Projeto

O **Coffee RH** foi desenvolvido como projeto acadêmico com o objetivo de simular um sistema de controle de ponto utilizado em empresas.

O sistema permite que usuários façam login, acessem seus serviços e registrem seu ponto de forma rápida e organizada.

---

## 🚀 Funcionalidades

✔ Sistema de Login  
✔ Página de Serviços  
✔ Visualização de Perfil  
✔ Registro de Ponto Virtual  
✔ Página institucional "Sobre Nós"

---

## 📷 Telas do Sistema

### 🔐 Login
Tela de autenticação do usuário para acesso ao sistema.

### 🏠 Serviços
Área principal com acesso às funcionalidades disponíveis.

### 👤 Perfil
Exibição das informações do usuário logado.

### ⏰ Registro de Ponto
Sistema para registrar o horário de trabalho.

### 📖 Sobre Nós
Página institucional apresentando a missão e história do projeto.
